## 📋 项目简介

一个极简主义的实时通信工具，基于UDP协议实现。不需要繁琐的账号注册，**设置相同的暗号即可开始聊天**，让通信回归简单本质。

### 🎯 核心玩法

> 设置相同暗号的人可以一起聊天，像对暗号一样简单有趣！

## 🧠 开发理念

### 💪 造轮子的艺术

> **"尽量使用自己写的代码，尽力重复造轮子，第三方库能不用就不用"**

这不是简单的重复造轮子，而是一个学习的过程：

```
能不能造轮子？ → 学习能力 → 实践
      ↓                    
想不想造轮子？ → 选择能力 → 创新
      ↓                    
   🎯 目标：先解决能不能的问题
```

### 🎯 为什么选择Go？

- 📚 **学习驱动**：希望通过实践掌握Go技术栈
- 🚀 **开发体验**：简洁的语法，高效的开发
- 🔧 **实用主义**：部署简单，性能优秀
